================================================================================
🤖 QU'EST-CE QUE ROBOTS.TXT ?
================================================================================

robots.txt est un fichier texte qui dit aux moteurs de recherche (Google, Bing, etc.)
comment traiter votre site.

FONCTION :
- Empêcher l'indexation (ne pas apparaître dans Google)
- Bloquer les robots
- Protéger les parties sensibles du site

NOTRE CAS :
- Nous avons configuré robots.txt pour BLOQUER TOUS les robots
- Cela empêche Google/Bing de référencer votre site de test

================================================================================
📍 OÙ PLACER LE FICHIER ROBOTS.TXT
================================================================================

Votre structure doit être :

📁 judo-club-des-avirons/
├── 📄 robots.txt          ← À la RACINE (même niveau que index.html)
├── 📄 index.html
├── 📄 club.html
├── ... autres pages
├── 📁 css/
│   └── 📄 style.css
└── 📁 images/
    └── 📄 logo-jca.png

================================================================================
✅ CONTENU DU ROBOTS.TXT
================================================================================

User-agent: *
Disallow: /

Cela signifie :
- User-agent: * = "Pour TOUS les robots"
- Disallow: / = "Interdisez l'accès à tout le site"

RÉSULTAT :
✓ Google ne référencera PAS votre site
✓ Bing ne l'indexera PAS
✓ Les autres moteurs l'ignoreront
✓ Mais les gens peuvent TOUJOURS accéder au lien direct

⚙️ COMMENT UPLOADER ROBOTS.TXT SUR GITHUB
================================================================================

OPTION A : Via l'interface GitHub (SIMPLE)
──────────────────────────────────

1. Allez sur votre dépôt GitHub
2. Cliquez "Add file" → "Create new file"
3. Nommez-le : robots.txt
4. Collez ce contenu :

   User-agent: *
   Disallow: /

5. Cliquez "Commit new file"
6. ✅ robots.txt est maintenant sur GitHub !

OPTION B : Avec Git (si vous faites ça localement)
───────────────────────────────────────────────

1. Placez robots.txt dans votre dossier du site
2. Ouvrez le terminal
3. Tapez :
   
   git add robots.txt
   git commit -m "Add robots.txt to prevent indexation"
   git push

4. ✅ robots.txt est uploadé !
