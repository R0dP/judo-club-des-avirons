🚀 PROCHAINES ÉTAPES
================================================================================

1️⃣  TESTER LE SITE LOCALEMENT
   • Ouvrez "index.html" dans votre navigateur
   • Explorez toutes les pages
   • Vérifiez les liens

2️⃣  INTÉGRER VOS IMAGES
   • Créez un dossier "images" dans le même répertoire
   • Téléchargez vos photos (dojo, entraîneurs, événements)
   • Remplacez les placeholders par vos images
   ➜ Voir README.md pour les instructions détaillées

3️⃣  PERSONNALISER LE CONTENU
   • Modifiez les textes selon vos besoins
   • Ajoutez/modifiez les articles du blog
   • Mettez à jour les horaires et tarifs

4️⃣  HÉBERGER EN LIGNE (Optionnel mais recommandé)
   • Choisir un hébergeur (OVH, 1&1, Hostinger, etc.)
   • Enregistrer un domaine (www.judoclubavirons.com)
   • Uploader les fichiers via FTP
   ➜ Voir README.md pour plus de détails

🎨 PERSONNALISATION
================================================================================

COULEURS (faciles à changer) :
   --primary-blue: #185FA5     (bleu principal)
   --dark-blue: #042C53        (bleu foncé)
   --accent-gold: #BA7517      (or/accent)

Editez css/style.css pour changer tous ces éléments à la fois.

CONTENU (très facile à changer) :
   • Ouvrez n'importe quel fichier .html avec un éditeur texte
   • Modifiez directement le contenu
   • Sauvegardez et rafraîchissez la page

IMAGES :
   • Créez un dossier "images" dans le répertoire du site
   • Placez vos photos dedans
   • Remplacez les placeholders comme expliqué dans README.md
