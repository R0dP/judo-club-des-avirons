✅ CHECKLIST ÉTAPE PAR ÉTAPE
================================================================================

AVANT DE COMMENCER :
☐ Vous avez téléchargé tous vos fichiers du site
☐ Les fichiers sont bien organisés (HTML + dossier css/)
☐ Vous avez une connexion internet
☐ Vous avez un email pour créer le compte

ÉTAPE 1 - CRÉER ACCOUNT GITHUB :
☐ Allez sur https://github.com
☐ Cliquez "Sign up"
☐ Complétez le formulaire avec :
  ☐ Email valide
  ☐ Mot de passe fort
  ☐ Nom d'utilisateur (ex: votrenom-reunion)
☐ Vérifiez votre email
☐ Connectez-vous

ÉTAPE 2 - CRÉER UN DÉPÔT :
☐ Cliquez le "+" en haut à droite
☐ Sélectionnez "New repository"
☐ Remplissez les champs :
  ☐ Repository name : "judo-club-des-avirons"
  ☐ Description : "Site web du Judo Club des Avirons"
  ☐ Visibility : "Public" (IMPORTANT!)
☐ Cliquez "Create repository"

ÉTAPE 3 - UPLOADER LES FICHIERS :
☐ Cliquez "Add file" → "Upload files"
☐ Uploadez tous ces fichiers :
  ☐ index.html
  ☐ club.html
  ☐ equipe.html
  ☐ horaires.html
  ☐ tarifs.html
  ☐ histoire.html
  ☐ actualites.html
  ☐ contact.html
  ☐ css/ (dossier entier avec style.css)
☐ Cliquez "Commit changes"

ÉTAPE 4 - ACTIVER GITHUB PAGES :
☐ Cliquez "Settings" dans votre dépôt
☐ Cherchez "Pages" à gauche
☐ Sélectionnez "main" comme branch
☐ Sélectionnez "/ (root)" comme dossier
☐ Cliquez "Save"
☐ ATTENDEZ 1-2 MINUTES

ÉTAPE 5 - TESTER :
☐ Cherchez le lien généré (format: https://votreusername.github.io/judo-club-des-avirons/)
☐ Cliquez le lien
☐ Vérifiez que le site s'affiche correctement
☐ Testez quelques pages
☐ Testez sur mobile (F12)

ÉTAPE 6 - PARTAGER :
☐ Copiez votre lien GitHub Pages
☐ Envoyez-le aux responsables du club
☐ Demandez leurs retours
☐ Notez les améliorations demandées

🚨 EN CAS DE PROBLÈME
================================================================================

Le site ne s'affiche pas :
1. Attendez 3-5 minutes après l'activation
2. Rafraîchissez la page (Ctrl+F5)
3. Vérifiez que le dépôt est PUBLIC
4. Vérifiez que "index.html" est dans le dépôt

Les CSS ne se chargent pas :
1. Vérifiez que le dossier "css/" existe
2. Vérifiez que "style.css" est dedans
3. Vérifiez les noms exacts (majuscules/minuscules)
4. Rafraîchissez la page (Ctrl+F5)

GitHub Pages n'apparaît pas dans Settings :
1. Assurez-vous que le dépôt est PUBLIC
2. Scrollez dans Settings pour trouver "Pages"
3. Attendez un peu, il peut y avoir un délai
