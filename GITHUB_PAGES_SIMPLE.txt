🚀 GITHUB PAGES - VERSION ULTRA-SIMPLE (5 ÉTAPES)
================================================

Voici les 5 étapes pour mettre votre site en ligne GRATUITEMENT et SIMPLEMENT.

================================================================================
ÉTAPE 1 : CRÉER UN COMPTE GITHUB (5 MINUTES)
================================================================================

1. Allez sur : https://github.com
2. Cliquez "Sign up" (en haut à droite)
3. Entrez votre email
4. Créez un mot de passe
5. Choisissez un nom d'utilisateur (ex: "judo-club-reunion")
6. Vérifiez votre email
7. ✅ VOUS AVEZ UN COMPTE GITHUB !

================================================================================
ÉTAPE 2 : CRÉER UN DÉPÔT (1 MINUTE)
================================================================================

Sur votre page d'accueil GitHub :

1. Cliquez le "+" en haut à droite
2. Sélectionnez "New repository"
3. Remplissez :
   
   Repository name : judo-club-des-avirons
   Description : Site web du Judo Club des Avirons
   Visibility : PUBLIC ✅ (important!)
   
4. Cliquez "Create repository"

✅ VOTRE DÉPÔT EST CRÉÉ !

================================================================================
ÉTAPE 3 : UPLOADER VOS FICHIERS (2 MINUTES)
================================================================================

Sur la page de votre dépôt :

1. Cliquez "Add file" → "Upload files"

2. DÉPOSEZ TOUS VOS FICHIERS (drag and drop) :
   
   - index.html
   - club.html
   - equipe.html
   - horaires.html
   - tarifs.html
   - histoire.html
   - actualites.html
   - contact.html
   - css/style.css (mettez le dossier CSS en entier)
   
   (Les autres fichiers .txt sont optionnels)

3. En bas, cliquez "Commit changes"

✅ VOS FICHIERS SONT UPLOADÉS !

================================================================================
ÉTAPE 4 : ACTIVER GITHUB PAGES (1 MINUTE)
================================================================================

Sur la page de votre dépôt :

1. Cliquez "Settings" (onglet gris en haut)

2. À gauche, trouvez "Pages" (environ au milieu)

3. Sous "Source" :
   - Branch : "main" (ou "master")
   - Folder : "/ (root)"

4. Cliquez "Save"

✅ ATTENDEZ 1-2 MINUTES...

GitHub vous affichera :
"Your site is published at : https://votreusername.github.io/judo-club-des-avirons"

================================================================================
ÉTAPE 5 : PARTAGEZ VOTRE LIEN ! (30 SECONDES)
================================================================================

Votre site est maintenant EN LIGNE ! 🎉

Copiez ce lien et envoyez-le aux responsables du club :

   https://votreusername.github.io/judo-club-des-avirons/

Remplacez "votreusername" par votre vrai nom GitHub.

================================================================================
✏️ POUR MODIFIER VOTRE SITE PLUS TARD
================================================================================

Si vous voulez changer quelque chose :

1. Allez sur votre dépôt GitHub
2. Cliquez sur le fichier à modifier
3. Cliquez le crayon (Edit)
4. Modifiez et cliquez "Commit changes"
5. Le site se met à jour automatiquement (1-2 min)

================================================================================
❓ PROBLÈMES COURANTS
================================================================================

❌ Le site ne s'affiche pas ou est en blanc
→ Attendez 2-3 minutes après l'activation
→ Rafraîchissez la page (Ctrl+F5)
→ Vérifiez que "index.html" est bien uploadé

❌ Les CSS ne se chargent pas
→ Vérifiez que le dossier "css/" existe
→ Vérifiez que "style.css" est dedans

❌ GitHub Pages n'apparaît pas dans Settings
→ Assurez-vous que le dépôt est PUBLIC
→ Scrollez pour trouver "Pages" dans Settings
